Carta Correio

Este plugins permite a troca de cartas entre os jogadores e um inventário específico para as cartas.

Recursos:

  • ✅ Versão do minecraft 1.21.7
  • ✅ Comando para o envio da carta: /carta (Nome do Jogador)
  • ✅ Comando para acessar inventário de cartas: /correio
  • ✅ Totalmente compatível com Spigot
